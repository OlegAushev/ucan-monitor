# Установка

**Системные требования**: 
* Ubuntu 22.04
* Библиотека GLFW:

```bash
sudo apt install libglfw3
```

* Библиотека GLIBCXX_3.4.32:

```bash
sudo add-apt-repository ppa:ubuntu-toolchain-r/test
sudo apt install g++-13
```

# Запуск

```bash
cd <path-to-ucan-monitor-folder>/ucan-monitor/bin
./ucan-monitor
```
